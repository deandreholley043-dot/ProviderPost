## 📝 JOB DESCRIPTION

### Overview

We have a **complete, production-ready Next.js application** that's 100% built and tested. We need a developer to **deploy it to Vercel** with environment variables configured.

This is a **simple server deployment job** - just get the code running on Vercel with the right settings.

1. **Complete Next.js application** (ready to deploy)
2. **GitHub repository** (code is already there)
3. **Environment variables template** (you just fill in values)
4. **All dependencies** (package.json ready to go)
5. **Documentation** (everything is documented)

---

## 🎯 YOUR TASKS (Simple)

### Task 1: Create Vercel Account
- [ ] Go to https://vercel.com
- [ ] Sign up (free account)
- [ ] Connect GitHub account

### Task 2: Import Project to Vercel
- [ ] Click "Add New" → "Project"
- [ ] Select GitHub repository
- [ ] Click "Import"

### Task 3: Configure Environment Variables
- [ ] In Vercel settings, add these variables:
```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_SITE_URL=
```

Note: Client will provide these values

### Task 4: Deploy
- [ ] Click "Deploy"
- [ ] Wait for build to complete (2-3 minutes)
- [ ] Verify site is live

### Task 5: Test
- [ ] Open the live URL
- [ ] Verify pages load correctly
- [ ] Check that no errors appear
